# Free-water elimination for The University of California San Francisco Preoperative Diffuse Glioma MRI dataset

This repository includes code that analyzes the data available at https://www.cancerimagingarchive.net/collection/ucsf-pdgm/.

The code will download data available at: https://figshare.com/articles/dataset/Free-water_elimination_tractometry_from_the_The_University_of_California_San_Francisco_Preoperative_Diffuse_Glioma_MRI_dataset/30402736

If you reuse the data that , please follow the instructions for Citations & Data Usage Policy that are available with the original dataset:

Data Citation: Calabrese, E., Villanueva-Meyer, J., Rudie, J., Rauschecker, A., Baid, U., Bakas, S., Cha, S., Mongan, J., Hess, C. (2022).
